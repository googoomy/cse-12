Lab 1: Intro to Logic Simulation

------------
DESCRIPTION

In this lab, the user is supposed to change the input with the four switches on the first page to see the output of various simulations.

------------
FILES

-
Lab1.lgi

This file is ran through MML to view the logic simulations.

------------
INSTRUCTIONS

Lab1.lgi is intended to be ran through Multimedia Logic (MMLogic or MML). When the user presses the play button, 
they have access to four switches on the first page that changes the output of three different simulations.
